SORRY CARD — QUICK SETUP

1) Open script.js and edit only the CONFIG object at the top.
2) Put photos inside assets/images named photo1.jpg ... photo6.jpg, or change their paths in CONFIG.photos.
3) Put background music at assets/music/background.mp3.
4) Optional tiny popup sound: assets/sounds/chime.mp3.
5) Double-click index.html. No install or server is required.

DEPLOY
GitHub Pages: Push the folder contents to a repository > Settings > Pages > Deploy from branch > main/root.
Netlify: Drag the sorry-card folder into Netlify Drop.
Vercel: Import the GitHub repository and deploy as a static site with no build command.
